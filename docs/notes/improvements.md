## 除外項目

以下は実装対象外とする：

| 項目 | 理由 |
|------|------|
| 設定の永続化 | 都度開く使い方がメインのため不要 |
| キーボードショートカット | マウス操作がメインのため優先度低 |
| レスポンシブ対応 | PC専用アプリのため不要 |
| PiP | Twitch iframeの制限あり |
| PWA対応 | PC専用アプリのため不要 |

## 今後の方針

現時点で主要機能は全て実装済み。追加改修は以下の場合のみ検討：

1. パフォーマンス問題が発生した場合 → React.memo最適化
2. バグ修正の効率化が必要な場合 → テスト追加
3. ユーザーからの要望があった場合 → 個別対応
