# プロジェクト概要

Youtube/Twitch マルチビューア

## 何を作るか

既存のバニラ JS で作成された Youtube/Twitch マルチビューアを React で再実装し、保守性と拡張性を向上させる。

## 技術スタック

- 言語: TypeScript
- 主要ライブラリ/フレームワーク: React + Vite + Tailwind
- 主要ライブラリ
  - Youtube IFrame API
  - Twitch Embed

## 主な機能

### 複数配信の同時視聴
- YouTube・Twitch の配信を同時に表示
- 動画数に応じた自動グリッドレイアウト
- 各動画の個別削除・全動画リセット

### チャット機能
- 各配信のチャットをサイドパネルで表示/非表示切り替え
- YouTube: ライブチャット埋め込み、右クリックで別ウィンドウ表示
- Twitch: ライブチャット・VODチャットリプレイ対応

### 再生制御（YouTube のみ）
- ±5秒/10秒のシークボタンで配信のズレを調整
- 現在の再生時間をリアルタイム表示
- ミュート/ミュート解除

### URL 入力・プラットフォーム検出
- YouTube: watch, youtu.be, live, embed 形式に対応
- Twitch: チャンネル URL、VOD URL に対応
- ライブ配信/アーカイブの手動切り替えオプション

### UI/UX
- ダークモード固定
- ウェルカム画面（機能紹介）
- モーダルウィンドウによる動画追加操作
